About
=====
TaliaBeeIO is a [Python](https://www.python.org/) interface to TaliaBee API. TaliaBeeIO can be used to monitor and to control the IO of TaliaBee box.

Requirements
============
- Python 3.x
- Python Requests module

Installation
============

```bash
pip3 install taliabeeio
```
